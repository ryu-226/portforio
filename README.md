# めしガチャ
### サービスURL : https://www.meshigacha.com/
<img src="app/assets/images/ogp_default.png" alt="Image" width="800">

## ■ サービス概要 
節約しながら外食を楽しみたい！予算内で楽しむ“ランチ額抽選”アプリ。<br>
その月の外食したい日数と月の予算を設定することでその日使える飲食代をランダムで抽選します。<br>
抽選される金額の1日の最低金額と最高金額も自分で設定できます。<br>
また履歴ページで実際に使用した金額を入力することで、抽選金額との差額をいつでも確認し計画的に節約することができます。
<br>
<br>

## ■ このサービスへの思い・作りたい理由 
自分は食べることが好きでメニューにおいしそうな料理があるとつい値段を気にせずに頼んでしまい、食費がかさんでしまう傾向にありました。<br>
そこでこのようなアプリがあれば、抽選された金額内でメニューを決定しなければいけないという義務感が発生し食費を抑えることができると思いました。<br>
限られた予算の中で工夫してランチをたのしむことで達成感も味わえます。<br>
またその日抽選される金額をランダムにすることでドキドキ感があり、履歴ページで日々振り返りながら毎日楽しく計画的に節約ができると思います。
<br>
<br>

## ■ ユーザー層について 
- 節約を意識している且つ外食の機会が多い方
- 節約は意識しているが現状の収支を正しく理解できておらず、ついつい無計画になってしまう方
- 日々のお店選びがマンネリ化していて冒険したい方

  <br>

## ■ サービスの差別化ポイント・推しポイント
本アプリは金額をランダムで決めるという「不確実さを楽しむユニークさ」や、ユーザーに予算内で工夫し楽しんでもらう自主性やゲーム性があります。<br>
多くの節約アプリは記録型ですが、本アプリは「予算を守るための仕組みを先に設計」する点が新しく、使いすぎの抑止につながります。<br>
また毎日の抽選がくじ引きや占いのような感覚でワクワクでき、アプリを開くのが習慣化しやすいです。
<br>
<br>
